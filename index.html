  • 신규 문의자 상담 및 가입 안내

  • 신규 안내 페이지 24시간 운영
  • 상담 예약 자동 연결 (Cal.com)
  • 유튜브 예약 업로드
  • SNS 자동 배포
  • 가입 후 온보딩 자료 자동 발송

⚠️ 벨라에게 올려야 하는 것

  • 새로운 정책 관련 결정
  • 글로벌 신규 파트너 연결
  • 판단이 어려운 특수 상황

❓ 예상 Q&A — 리더 불안 해소

Q. "잘 못하면 어떡하죠?"
처음부터 잘할 필요 없어요. 상담 스크립트 드릴 거고, 노션에 다 정리해둘게요. 따라만 하면 돼요. 저도 처음엔 다 몰랐어요.
Q. "모르는 질문을 받으면요?"
노션 FAQ 확인 → 그래도 모르면 리더 단톡방에 물어보기 → 그래도 안 되면 벨라한테. 이 순서로 오면 돼요.
Q. "시간이 많이 걸릴 것 같아요"
시스템 갖춰지면 하루 1~2시간으로 운영 가능해요. 지금은 세팅 기간이라 조금 더 걸릴 수 있지만, 한 달만 투자하면 그다음부턴 훨씬 편해져요.
Q. "팀원들이 저한테 안 물어보면요?"
제가 팀원들한테도 공지할게요. '앞으로 리더한테 먼저 연락하세요'라고요. 구조를 제가 만들어드릴게요.

PART 5 · 리더 상담 스크립트

리더 전용
1단계 · 오프닝 (2분)
"안녕하세요! 연락 주셨군요 😊 혹시 어떻게 알게 되셨어요?"

→ 유튜브: "어떤 영상 보셨어요? 비용 공개 영상이요? 맞아요 실제예요 ㅎㅎ"
→ 지인 소개: "아 OOO님 통해서 오셨군요!"
2단계 · 니즈 파악 (3분)
"여행 자주 다니세요? 지금 여행할 때 가장 불편하거나 아깝다고 느끼는 게 뭐예요?"

→ 비용이 아깝다: "저도 그랬어요. 비용 비교 보여드릴게요" → 노션 비교 페이지 공유
→ 시간이 없다: "직구로 예약하면 훨씬 빠르거든요"
→ 여행을 못 간다: "멤버십 있으면 갑자기 저렴한 일정도 잡을 수 있어요"
3단계 · 솔루션 제시 (5분)
"제가 실제로 [본인 경험] 했는데요, 직구로 예약하면 일반이랑 이렇게 달라요"

→ 비용 비교 화면 보여주기
→ 벨라 유튜브 비용 공개 영상 링크 전달
→ "100만원대로 크루즈 가는 거 영상에 다 나와 있어요"
4단계 · 가입 안내 (3분)
"가입은 어렵지 않고요, 구독료 내시면 바로 예약 가능해요. 지금 바로 하실 수 있어요."

→ 가입 링크 전달
→ "궁금한 거 있으면 언제든 카톡 주세요!"
💡 못 정하고 끊으려 할 때
"결정 급하게 안 하셔도 되는데요, 제가 안내 자료 보내드릴게요. 천천히 보시고 연락 주세요 😊"

→ 노션 안내 페이지 링크 전달
